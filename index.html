About the Snap for Windows (SnapW) Team

We are a group of students at Brigham Young University working with Intel on an open-source telemetry project called Snap. Snap was created in July of 2015 with the intent of making a product that provided a single API for collecting telemetry data from Unix-based systems. Prior to our project, Snap did not function on Windows systems. We wrote a Microsoft Installer for Snap, documented the build and installation process, and created three Snap collector plugins.
